Clutch Client
=============

This is the command-line client interface for native/HTML hybrid products of
Clutch.io.

Note: You do not need to use this if you're only using Clutch A/B testing.


Commands
========

* startapp - Starts a new application
* startscreen - Starts a new screen for your application
* upload - Uploads a new version of your application to your Clutch.io service
* dev - Enters into development mode for your Clutch application
